# QuickStock ⚡📦

### QuickStock is a basic system designed for general product management in stores.

## 📌 Main Features

### 📦 Product Management
- Register new products.
- List all products.
- Check availability.

## 🛠️ Technologies and Tools Used

### ⚙️ Backend
- C#
- ASP.NET Core
- Entity Framework Core

### 🎨 Frontend
- HTML
- CSS
- JavaScript

### 🛢️ Database
- SQLite
- Entity Framework Core (ORM)

### 💾 Code Versioning
- Git
- GitHub
